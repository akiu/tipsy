Tipsy

MVW PHP framework

View on GitHub Download .zip Download .tar.gz

Tipsy is an MVW (Model, View, Whatever) PHP framework inspired by AngularJS. It provides a very lightweight, easy to use framework, capable of handling most tasks.

This is a dev version. I am currently pulling in features from Cana and building unit tests as I go.

Example Usage

See Examples for more detailed examples. See Documentation for more information.

index.php
require_once 'Tipsy.php';
use Tipsy\Tipsy;

$tipsy = new Tipsy;

$tipsy->router()
    ->when('hello', function($Scope, $View) {
        $Scope->user = 'Devin';
        $View->display('hello');
    })
    ->otherwise(function() {
        echo '404';
    });

$tipsy->start();
hello.phtml
<h1>Welcome, <?=$user?>!</h1>
