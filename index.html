tipsy

Github Examples Docs

Tipsy is an MVW (Model, View, Whatever) PHP micro framework inspired by AngularJS. It provides a very lightweight, easy to use framework for websites and rest apis.


Template Example

> index.php
$tipsy->router()
    ->home(function($Scope, $View) {
        $Scope->user = 'Mai Tai';
        $View->display('hello');
    });

> hello.phtml
<h1>Hello <?=$user?>!</h1>



API Example

> index.php
$tipsy->router()
    ->delete('api/maitai/:id', function($Params) {
        echo json_encode([message => $Params->id]);
    });

> DELETE /api/maitai/1
{"message": 1}




Installation

To install using composer use the command below. For alternate installation methods see Installation.

composer require arzynik/Tipsy
