Tipsy is an MVW (Model, View, Whatever) PHP framework inspired by AngularJS. It provides a very lightweight, easy to use framework, capable of handling most tasks.

This is a dev version. I am currently pulling in features from Cana and building unit tests as I go.

Example Usage

See Documentation for more information.

hello.php
require_once 'Tipsy.php';
use Tipsy\Tipsy;

$tipsy = new Tipsy;

$tipsy->router()
    ->home(function($View) {
        $View->display('home');
    });
    ->when('user', function($Scope, $View) {
        $Scope->user = 'Devin';
        $View->display('user');
    })
    ->otherwise(function() {
        echo '404';
    });

$tipsy->start();
user.phtml
<div class="content">
    <h1>Welcome <?=$user?>!</h1>
</div>
